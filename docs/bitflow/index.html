Valor

Excel vs BitFlow Campo

Excel clasico

  • Sin flujo nativo mobile para campo.
  • Adjuntos fuera de la celda y contexto.
  • Riesgo alto de perdida sin guardado consistente.

BitFlow Campo

  • UI mobile-first pensada para captura en sitio.
  • Adjuntos inline y evidencias en el mismo flujo.
  • Offline-first con salida a XLSX cuando haga falta.

Onboarding

Cómo empezar

  1. Abrir palette (Ctrl/Cmd+K o âš¡)

    En desktop usa Ctrl/Cmd+K. En mobile toca âš¡ para abrir acciones premium.

  2. Nuevo registro + pegar tabla

    Crea una fila base, pega TSV/CSV, confirma preview y aplica en lotes.

  3. FlowBot + macros + modo campo

    Ejecuta comando con preview, guarda macro y activa modo campo para capturar sin ruido.

  4. Undo y Exportar

    Si algo no cierra, deshace en un toque. Luego exporta en XLSX o PDF.
