# Sales Demo Script (2 minutos)

## Objetivo
Mostrar valor de BitFlow en un flujo real: captura, gestion y entrega sin backend obligatorio.

## Guion sugerido
1. Problema (15s)
- "Hoy muchos equipos de campo trabajan con planillas dispersas, evidencias sueltas y tiempos de cierre lentos."

2. Propuesta BitFlow (20s)
- "BitFlow unifica planilla + evidencia en un flujo local-first: funciona aun sin red y exporta entregables listos."

3. Demo operativa (60s)
- Entrar a `/app`.
- Crear una planilla nueva.
- Editar 2-3 celdas.
- Mostrar que la accion principal de guardar es clara.
- Abrir export y generar archivo.
- Mostrar Premium/Agent como extension comercial del flujo.

4. Cierre comercial (25s)
- "Resultado: menos friccion operativa, mejor trazabilidad y entregables mas rapidos."
- "Implementacion simple en web, sin exigir backend para iniciar."

## Frases de apoyo (cortas)
- "Menos friccion en campo, mas velocidad de cierre."
- "Local-first real: continuidad operativa sin depender de internet."
- "Exportacion profesional lista para cliente."
