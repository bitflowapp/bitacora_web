## Long description
BitFlow ayuda a equipos de campo a registrar relevamientos de forma rapida y clara.  
Crea planillas en segundos, adjunta evidencia por celda (fotos y audio), captura ubicacion cuando la necesitas y exporta respaldos para mantener continuidad operativa.

Pensado para uso diario en terreno:
- Flujo simple para crear, completar y guardar planillas.
- Adjuntos por celda para contexto tecnico real.
- Diagnostico integrado para soporte sin herramientas externas.
- Importacion y exportacion de backup ZIP para mover trabajo entre equipos.

BitFlow prioriza almacenamiento local y control del usuario sobre sus datos.
