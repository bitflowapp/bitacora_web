# Onboarding BitFlow (5 minutos)

## 1) Crear planilla con plantilla (1 min)
1. Abrí BitFlow Web: https://marcoluna-nqn.github.io/bitacora_web/
2. Entrá a **Crear planilla**.
3. Elegí una plantilla: Base, Resistividades, Inventario o Checklist.
4. Confirmá el nombre y comenzá.

## 2) Usar Modo Campo (+Registro) (1 min)
1. Tocá **+Registro** en la planilla.
2. Cargá datos clave del evento.
3. Adjuntá foto, GPS/timestamp y nota cuando aplique.
4. Guardá para sumar el registro a la grilla.

## 3) Editar rápido (1 min)
- **Ctrl/Cmd + K**: búsqueda o comandos rápidos.
- **Ctrl/Cmd + S**: guardar.
- **Ctrl/Cmd + F**: buscar dentro de la planilla.

## 4) Exportar y compartir (1 min)
1. Abrí **Exportar**.
2. Elegí formato: **XLSX** o **PDF**.
3. Si necesitás adjuntos, activá salida **ZIP**.
4. Compartí por correo o apps de compartir (por ejemplo WhatsApp según plataforma).

## 5) Offline + sync (1 min)
1. Si no hay señal, seguí cargando: queda guardado localmente.
2. Cuando aparece **Pendiente de sync**, el registro está en cola para envío.
3. Al volver internet, BitFlow reintenta sincronizar automáticamente.
4. Si hace falta, reintentá manualmente desde la cola offline.

## iPhone / iOS (PWA)
1. Abrí BitFlow en Safari.
2. Tocá **Compartir**.
3. Elegí **Agregar a pantalla de inicio**.
